tf_codage.models - base deep learning models¶

tf_codage.models.common - long-text transformers models¶

Implement models for various tasks. Most of the models are sub-classed of the huggingface transformers models, so please checkout their docs for more information

models.TFCamembertForSentenceEmbedding(…)

Camembert model for calculating sentence embeddings through an average of word embeddings.

models.FullTextBert(*args, **kwargs)

Camembert model that can handle long documents by splitting them into separate token batches.

models.FullTextConfig([pool_size, …])

Extra configuration for BERT for long texts.

models.BertForMultilabelClassification(…)

Bert for sequence classification with sigmoid activation in the output layer

models.CamembertForMultilabelClassification(…)

Camembert for sequence classification with sigmoid activation in the output layer

models.MeanMaskedPooling(*args, **kwargs)

Mean pooling with strides and mask

models.MaxMaskedPooling(*args, **kwargs)

Max pooling with strides and mask

models.PoolingClassificationHead(*args, **kwargs)

Classification head with pooling layer.

tf_codage.utils - common utilities¶

Set of helper functions to work with files, standard output etc. They are not specific to machine learning or medical informatics.

utils.TeeStream(*streams)

utils.print_console(*args, **kwargs)
