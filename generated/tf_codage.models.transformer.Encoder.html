tf_codage.models.transformer.Encoder¶

class tf_codage.models.transformer.Encoder(*args, **kwargs)[source]¶

Encoder part of the transformer architecture.

__init__(num_layers, d_model, num_heads, dff, maximum_position_encoding, rate=0.1, input_vocab_size=None)[source]¶

Creates an encoder layer.

Encoder layer takes a sequence of token ids (or embeddings, if input_vocab_size is None) and returns a sequence of contextualised embeddings.

Parameters

  • num_layers – number of encoder layers

  • d_model – number of units in hidden layers (and embedding size)

  • num_heads – number of attention heads

  • dff – number of units in feedforward layers

  • maximum_position_encoding – maximum input sequence length

  • rate – dropout rate

  • input_vocab_size – size of input vocabulary (or None in which case the inputs must be already embeddings).

call(x, training, mask)[source]¶

Run the encoder layer with the inputs.

Parameters

  • x – sequence of token ids or embeddings (if input_vocabulary_size is None)

  • training – use True if used during training and False for evaluation

  • mask – mask for padding tokens
