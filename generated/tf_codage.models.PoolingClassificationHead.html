tf_codage.models.PoolingClassificationHead¶

class tf_codage.models.PoolingClassificationHead(*args, **kwargs)[source]¶

Classification head with pooling layer.

The type of pooling can be ‘mean’, ‘max’ or an instance of a custom pooling mechanism subclassed from Layer.

__init__(config, *args, **kwargs)[source]¶
call(inputs, mask)[source]¶

This is where the layer’s logic lives.

Note here that call() method in tf.keras is little bit different from keras API. In keras API, you can pass support masking for layers as additional arguments. Whereas tf.keras has compute_mask() method to support masking.

Parameters

  • inputs – Input tensor, or list/tuple of input tensors.

  • **kwargs – Additional keyword arguments. Currently unused.

Returns

A tensor or list/tuple of tensors.
